# CraveKart Partner App

A modern mobile application for Restaurant Owners and Delivery Partners.

## Features
- **Restaurant Owners**: Dashboard, Menu Management, Incoming Orders, Analytics.
- **Delivery Partners**: Status Toggle, Request Management, Live Tracking, Earnings.

## Tech Stack
- Expo / React Native
- Expo Router
- Lucide/Ionicons Icons
- Linear Gradient
- Native Safe Area Context

## Setup
1. `cd partner-app`
2. `npm install`
3. `npm start`

## Theme
- Primary: Deep Purple (#4B0082)
- Secondary: Yellow (#FFC300)
